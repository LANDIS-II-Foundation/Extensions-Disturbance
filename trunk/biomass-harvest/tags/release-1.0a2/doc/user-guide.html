Introduction

The Biomass Harvest Extension for LANDIS-II behaves pretty much like Base Harvest, so consult the user guide for Base Harvest for further information. Biomass Harvest supports partial thinning of cohorts.

History

Version 1.0 (alpha 2 release)

  • Fixed bug with stand-spreading algorithms for site selection (CompleteStandSpreading and PartialStandSpreading). The initial augmentation to site selection to accommodate partial thinning crashed when a stand-spreading algorithm was specified in a prescription.

Input Files

The input file for Biomass Harvest is very similiar in format to the input file for Base Harvest, except for these differences:

  • The LandisData parameter requires a value of "Biomass Harvest".
  • When SpeciesList is specified for the CohortsRemoved parameter in a prescription, the user may specify a percentage after any individual age or range of ages. For example: 
            CohortsRemoved  SpeciesList
          acersacc      1-40 (50%)  50(65%)  65-70  71-107 ( 15% )  109
          ...
    The default percentage is 100%.
